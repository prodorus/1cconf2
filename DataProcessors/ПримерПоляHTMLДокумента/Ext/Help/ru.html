Пример использования элемента управления ПолеHTMLДокумента.

 

Обработка демонстрирует возможности элемента управления на примере работы со справочниками текущей конфигурации:
- отображает список справочников;
- позволяет открыть список справочника;

- позволяет добавить в справочник новый элемент;

- отображает выбранный справочник в виде HTML документа.

 

В модуле формы обработки демонстрируются различные приемы работы элементом управления и с макетами типа HTML-документ:

- программное наполнение элемента управления ПолеHTMLДокумента;

- использование макета при формировании HTML-документа;

- обработка событий в элементе управления ПолеHTMLДокумента.
