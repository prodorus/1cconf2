Эта форма предназначена для ввода сведений о номенклатурной позиции.

Закладка "Основные"

На этой закладке редактируются наиболее важные сведения о номенклатурной позиции.

Закладка "Картинка"

На этой закладке можно выбрать картинку, иллюстрирующую данную номенклатурную позицию.

Закладка "Цены"

На этой закладке можно ввести цену данной номенклатурной позиции для каждого типа цен, определенного в справочнике "Типы цен".

Закладка "Ед. измерения"

На этой закладке можно указать, в каких еще единицах измерения (кроме той, что указана на закладке "Основные"), можно указывать в документах количество по данной номенклатурной позиции.

Для целей учета количество будет автоматически пересчитываться в основную единицу измерения путем умножения на указанный коэффициент.
