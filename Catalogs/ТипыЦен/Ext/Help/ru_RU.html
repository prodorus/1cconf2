В этом справочнике хранится набор типов цен, по которым организация реализует номенклатуру.

Для каждого типа цен по каждой номенклатурной позиции цена может быть назначена независимо; в расходной накладной можно выбрать нужный тип цен, и для каждой номенклатурной позиции будет автоматически подставлена соответствующая цена.
